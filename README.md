# Java COMPLETO 2026 ☕
> Instrutor: Nelio Alves (DevSuperior)

Repositório para organizar estudos e projetos do curso mais completo de Java, focado em Programação Orientada a Objetos, Spring Boot e bancos de dados relacionais e NoSQL.

## 🎯 Objetivo
Dominar o desenvolvimento de aplicações modernas em Java como graduando em Engenharia de Software, cobrindo desde a base da linguagem até arquiteturas de microsserviços.
